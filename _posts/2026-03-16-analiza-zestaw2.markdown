# Zadanie 1 -- Dziedzina funkcji (podpunkt i)

Rozważamy funkcję:

f(x,y) = sqrt(x\^2 - y\^2)

## Warunek istnienia pierwiastka

Aby pierwiastek był określony, wyrażenie pod nim musi być nieujemne:

x\^2 - y\^2 ≥ 0

## Przekształcenie warunku

x\^2 ≥ y\^2

co jest równoważne:

\|x\| ≥ \|y\|

## Dziedzina funkcji

D = { (x,y) ∈ R² : x\^2 - y\^2 ≥ 0 }

czyli wszystkie punkty spełniające:

-\|x\| ≤ y ≤ \|x\|

## Interpretacja geometryczna

Granice dziedziny wyznaczają proste:

y = x\
y = -x

Dziedzina to obszar pomiędzy tymi prostymi, w którym spełniony jest
warunek \|x\| ≥ \|y\|.\
Graficznie tworzy to dwa symetryczne obszary otwierające się w lewo i w
prawo względem osi x.

## Podsumowanie

Dziedziną funkcji są wszystkie punkty płaszczyzny spełniające warunek:

x\^2 - y\^2 ≥ 0

czyli punkty leżące pomiędzy prostymi y = x oraz y = -x (łącznie z tymi
prostymi).
